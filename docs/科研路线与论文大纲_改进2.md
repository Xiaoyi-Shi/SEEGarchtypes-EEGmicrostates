# 科研路线与论文大纲（修改版）

## 一、研究价值评估

### 1.1 这个研究问题有多大价值？

**结论先行：这是一个高度原创、具有明确发表价值的研究问题，但需要严格控制叙事边界，并将静息态主线与发作阶段扩展清晰分层。**

#### 支撑理由

**① EEG microstates 是成熟领域，但颅内对应物几乎空白**

EEG microstates 自 Lehmann 等人 1987 年提出以来已有大量研究[^1]，在静息态、认知和疾病研究中均有广泛应用[^2][^3]。然而，现有 microstate 研究几乎全部集中在头皮 EEG。颅内记录虽然具有更高的时空分辨率，但由于电极覆盖有限、病人间电极位置差异显著，跨病人比较始终是难点。你的研究通过“病人内 field-state 提取 → 共同空间 archetype 投影”的路线，正面解决了这一障碍。

**② SEEG 的全局场状态研究是真正的空白点**

颅内脑电（iEEG/SEEG）现有研究主要聚焦于局部场电位特征、癫痫网络传播、发作起始与高频活动等[^4][^5]。而在静息态 SEEG 中系统定义类似 microstate 的“全局场状态”，目前文献中几乎没有成熟工作。这意味着你的 Aim 1 本身就具备原创性和方法学价值。

**③ EEG-SEEG 跨模态同步的近零时滞发现具有机制意义**

近零时滞的宽同步峰（peak width ~41 ms）并不是平凡的阴性结果，而是一个机制性发现。它提示 EEG 的全局场状态与 SEEG 的全局场状态之间更可能存在共享驱动，而非稳定的单向领先-滞后关系。这一结果与“shared global drive”假说相吻合[^6]。

**④ Archetype 方法学具有跨领域推广价值**

将病人内模板投影到共同空间（Yeo17）后再进行 archetype matching，本身就是一种可迁移的分析框架。该策略不仅适用于 SEEG，也可推广至 DBS、ECoG 等具有电极覆盖异质性的颅内记录研究。

## 二、当前路线的不足与改进建议

### 2.1 方法学层面的待强化点

#### 问题一：为什么 4 个 archetype 的 group-level EEG maps 如此相似（r=0.982–0.999）？

这是当前最需要解释的结果之一。可能有三种解释：

- **可能 A（抹平效应）**：群体平均后的 EEG 条件图被不同病人的背景活动“抹平”。
- **可能 B（共享背景）**：不同 archetype 对应的 EEG 状态本质上相似，说明 EEG 对 SEEG 内部差异不敏感。
- **可能 C（方法学退化）**：archetype assignment similarity 中等，提示跨病人模板稳定性有限，投影后可能朝“平均模板”收缩。

**建议：**
- 增加 single-subject 层面的 archetype-conditioned EEG maps。
- 在 Discussion 中明确讨论这三种可能，不回避这一结果。

#### 问题二：近零时滞宽同步峰（41 ms）的解释需要更严格对照

宽同步峰可能来自：
- 真实的生理同步
- EEG 和 SEEG 的共同上游驱动
- 方法学平滑或 backfitting 带来的时间相关性

**建议：**
- 用不同带宽（如 1–20 Hz vs 1–40 Hz）做敏感性分析。
- 明确报告 backfitting 最短持续时长对峰宽的影响。
- 考虑使用替代 microstate 分割方法做鲁棒性验证。

#### 问题三：Archetype matching 强度中等，需要统计基准

0.348–0.536 这样的相似度数值本身缺少自然参照，因此需要一个 null 分布来解释。

**建议：**
- 通过置换病人内模板标签，建立随机基准。
- 报告观测值在 null 分布中的百分位。
- 这会显著增强 Aim 2 的说服力。

#### 问题四：SEEG field-state 的 K=4 选择需要更多验证

K=4 与经典 EEG microstate 文献一致，但这仍需数据驱动支撑。

**建议：**
- 增加 K=2..7 的敏感性分析。
- 同时报告 fit plateau、incremental gain、split-half stability。
- 说明 K=4 既有数据支持，也与文献传统对齐。

#### 问题五：样本量（N=20）对某些效应检测功效有限

20 例患者足以完成病人内分析，但某些 group-level 或 switching 结果可能功效不足。

**建议：**
- 在局限性中明确样本量对 switching 检测的限制。
- 如有可能，补充简单功效分析。

### 2.2 叙事层面的改进建议

#### 改进一：阴性结果应被积极论述

switching 阴性结果不应被简单表述为“没有发现”，而应与近零时滞同步结果一起解释：  
**这组结果共同支持 shared global drive，而不是单向状态传递。**

#### 改进二：强化与 fMRI-EEG microstate 文献的对比

fMRI-EEG microstate 关联是现有跨模态研究的主流[^7]。你的 SEEG 研究提供了更高时间分辨率的颅内证据，应在 Introduction 中明确这一定位。

#### 改进三：临床背景需要交代清楚

应明确说明：
- 数据来自癫痫外科评估患者；
- 静息态数据是否处于发作间期；
- 是否排除明显放电或严重伪迹片段。

## 三、完善后的科研路线与论文大纲

### 论文标题（建议）

**英文**：*Discrete Global-Field States in Human SEEG: Identification, Group Archetypes, and Near-Zero-Lag Synchrony with EEG Microstates*

**中文**：*人类 SEEG 中的离散全局场状态：识别、群体原型及其与 EEG 微状态的近零时滞同步*

---

### Abstract

**Background**: EEG microstates are discrete, quasi-stable scalp topographies reflecting large-scale brain state organization. Whether analogous global-field states exist in stereoelectroencephalography (SEEG) and how they relate to EEG microstates remains unknown.

**Methods**: We analyzed simultaneous resting-state EEG and SEEG recordings from 20 patients. Subject-level SEEG global-field states were extracted from bipolar peak maps using polarity-invariant clustering, with `K=4` retained for the main text. A supplementary `K=2..7` model-order evaluation assessed fit plateau, incremental gain, and stability. States were projected to a common Yeo17 network space to derive group archetypes. Synchrony between SEEG archetypes and EEG microstates was evaluated using fine-lag coupling at native 4-ms resolution.

**Results**: SEEG global-field states were reliably identified in all 20 patients (occupancy: 24–27%; dwell time: 96–104 ms). Four reproducible group archetypes emerged with distinct Yeo17 network loadings. EEG microstates and SEEG archetypes exhibited robust near-zero-lag synchrony (peak at 0 ms; 21/21 lag points significant; peak width ~41 ms), consistent with shared global drive rather than unidirectional state switching.

**Conclusions**: SEEG contains discrete global-field states that form reproducible group archetypes and synchronize with EEG microstates through near-zero-lag coupling. These findings establish a cross-modal framework for studying large-scale brain state organization using intracranial recordings.

## 2. Methods

### 2.1 参与者与数据采集
- 20 例接受 SEEG 监测的癫痫外科评估患者
- 同步采集 resting-state EEG（11 导联）和 SEEG（双极参考）
- 分析时间段：IDE_A 状态
- 排除标准：数据质量、有效双极道数量、EEG 覆盖完整性

### 2.2 EEG 预处理与 microstate 标注
- 1–40 Hz 带通、250 Hz 重采样、标准 19 导布局插值、平均参考
- 使用预训练 ModKMeans 模板（K=4）进行标注
- 输出 microstate 标签、GFP trace 和 GFP peaks

### 2.3 SEEG 预处理与双极道筛选
- 1–40 Hz 带通、250 Hz 重采样
- 仅保留两端接触点均归属同一 AAL3 区域的双极道
- 病人内 z-score 标准化

### 2.4 Subject-level SEEG global-field state 提取
- 全局峰值定义：跨双极道 RMS 的局部最大值
- 峰图提取：在峰值时刻提取瞬时双极空间构型
- 聚类：极性不变模板聚类
- 主分析保留 `K=4`
- 连续 backfitting：最短状态持续时长 30 ms
- `K=2..7` 的模型阶数评估用于补充验证：fit plateau、incremental gain、split-half stability 和 interpretability

### 2.5 Group archetype derivation in common space
- 共同空间：Yeo17 网络分区
- 投影：病人内模板 → Yeo17 网络均值 → 符号定向 → 跨病人 archetype matching
- 输出：archetype 模板、support、assignment similarity
- 说明：Yeo17 loading 为相对空间贡献，不代表绝对生理幅度

### 2.6 Fine-lag synchrony analysis
- 分析分辨率：250 Hz 原生时间轴
- Lag 扫描窗口：[-40, +40] ms，步长 4 ms
- 效应量：archetype 与 EEG microstate 的一致性指标
- Null 分布：circular-shift surrogate
- Group-level 推断：patient-first → sign-flip permutation

### 2.7 Archetype-conditioned EEG topography
- 将 archetype 标签对齐到 EEG 时间轴
- 计算每位患者每个 archetype 下的 EEG 平均 scalp map
- 汇总到 group-level
- 与 EEG microstate 模板的空间相似性比较
- 计算条件概率 P(EEG state = i | archetype = j)

### 2.8 Supplementary analyses
- GFP-global coupling
- GFP 控制后的 microstate / transition 效应
- SEEG → EEG switching 分析
- `1–20 Hz` vs `1–40 Hz` 敏感性分析
- `K=2..7` 模型阶数分析
- archetype similarity 的置换基准

## 2.9 IDE-A baseline vs seizure-stage comparison

### 2.9.1 研究目的与分析框架
本节为**扩展性分析**，用于评估在统一参考状态空间下，发作阶段相对于 IDE-A 基线是否存在状态偏移。  
**IDE-A 被定义为患者内基线状态**，并作为所有发作阶段数据的参考空间。

### 2.9.2 状态空间与比较原则
- **不在 SZ 阶段重新聚类**
- EEG microstate 与 SEEG archetype 均投影/标注到 **IDE-A-derived reference space**
- 保证各 stage 的状态编号和空间定义一致
- 所有比较均基于患者内配对差值，而非跨患者绝对值

### 2.9.3 统计单位与汇总方式
- 主分析使用 **patient-first summaries**
- 同一患者的多次发作，先在患者内按 stage 求均值
- 若某 stage 对应多个发作事件，则先做 patient-level averaging，再进行 group-level inference
- seizure-level 行仅用于轨迹展示和异质性描述
- 必须报告每个 stage 的有效患者数（denominator）

### 2.9.4 主效应量定义
\[
\Delta(stage) = \bar{x}_{patient, stage} - \bar{x}_{patient, IDE\_A}
\]

其中 \(x\) 可代表：
- EEG microstate occupancy / dwell time / occurrence
- SEEG archetype occupancy / dwell time / assignment similarity
- Yeo17 loading
- EEG-SEEG relationship 指标（如条件概率）

### 2.9.5 指标优先级
**一级主指标**
- EEG microstate occupancy
- EEG microstate dwell time
- SEEG archetype occupancy
- SEEG archetype dwell time

**二级结构指标**
- Yeo17 loading 的 stage-wise 变化
- archetype composition 的偏移

**三级关系指标**
- P(EEG state | archetype) 的 stage-wise 改变
- EEG-SEEG coupling 强度的变化

### 2.9.6 分层与敏感性分析
- seizure type（如 CPS / GTCS）分层
- repeated seizures within patient 的敏感性分析
- stage duration imbalance 的影响评估
- 若某些 stage 缺失，明确说明并进行完整病例分析与敏感性比较

### 2.9.7 解释边界
该分析描述的是**发作阶段相对于 IDE-A 的状态偏移**，不能单独用于证明癫痫发展机制或因果传播关系。其主要价值在于：  
1. 验证同一状态空间在发作阶段的可迁移性；  
2. 描述状态参数在 seizure evolution 中的重分配；  
3. 连接静息态机制结果与临床发作阶段现象。

### 2.9.8 与脚本输出的对应关系
- `scripts/07_seizure_stage_microstate_archetype_trajectory.Rmd`：仅负责 seizure-stage 自身的描述性图，不做 IDE-A baseline 推断。
- `scripts/08_ide_a_vs_seizure_stage_comparison.Rmd`：负责 `IDE-A -> pre/LVFA/SZ/post` 的 patient-first delta、forest plot、heatmap 与敏感性分析。
- `07` 关注：denominator、timing/processing QC、stage duration imbalance、projection confidence、stage-wise descriptive trajectories。
- `08` 关注：matched patient 数、primary occupancy/dwell delta、secondary Yeo17 delta、tertiary EEG-SEEG relationship delta、CPS/GTCS 与 repeated-seizure sensitivity。

### 2.9.9 当前主要结果与解读
**数据支持与 QC**
- IDE-A、pre、LVFA、SZ/ictal、post 的主分析均可形成患者内配对比较。
- 主要 EEG microstate 与 SEEG archetype 指标均有 20/20 名匹配患者完成 complete-case 分析。
- seizure-stage 数据存在重复发作，主要阶段约 65-67 个可用 seizure-stage 片段，因此主推断应以 patient-first delta 为准。
- LVFA 时长明显短于其他阶段：CPS 的 LVFA 中位数约 25.4 s，GTCS 的 LVFA 中位数约 27.6 s；因此 LVFA 中 dwell time 与 occurrence 指标需谨慎解释，occupancy 更适合作为一级主指标。

**EEG microstate 结果**
- 相对于 IDE-A，Microstate 3 的 occupancy 在所有 seizure stage 均呈升高方向：pre 约 +3.79 percentage points，LVFA 约 +4.20 percentage points，SZ/ictal 约 +6.18 percentage points，post 约 +6.71 percentage points。
- post 阶段 Microstate 3 的 mean dwell time 升高约 +50.0 ms，95% CI 约 +13.5 到 +86.6 ms。
- post 阶段 Microstate 2 occupancy 下降约 -3.15 percentage points，提示 postictal 阶段 EEG 状态分布从 M2 向 M3 偏移。
- 因此 EEG 层面的主结果可概括为：发作相关阶段，尤其 SZ/ictal 与 post 阶段，头皮 EEG microstate 更偏向 M3 表达。

**SEEG archetype 结果**
- SEEG archetype occupancy 的 stage-wise delta 整体较小，多数在 +/-2 percentage points 内，说明发作阶段未表现为 archetype 占比的大幅重排。
- 更明显的变化集中在 dwell time：pre 阶段 A1 约 -20.1 ms、A2 约 -10.7 ms；LVFA 阶段 A0 约 -10.0 ms、A1 约 -14.4 ms；SZ/ictal 阶段 A1 约 -20.6 ms、A3 约 -22.4 ms、A0 约 -9.2 ms。
- post 阶段 dwell time 转为升高，A0 约 +26.8 ms，A2 约 +26.3 ms，A3 约 +19.2 ms。
- 因此 SEEG 层面的主结果可概括为：发作前到发作期 SEEG 状态切换更快，而 postictal 阶段状态停留时间延长，提示深部网络动力学在发作后进入较慢的恢复状态。

**EEG-SEEG 关系结果**
- 条件概率 `P(EEG microstate | SEEG archetype)` 的变化主要指向 Microstate 3：多个 archetype 到 M3 的条件概率在 SZ/ictal 与 post 阶段升高。
- 最强方向包括 SZ 阶段 A3 -> M3 约 +7.08 percentage points，以及 post 阶段 A3 -> M3 约 +7.34 percentage points。
- 与此相对，A3 -> M2 在 SZ 与 post 阶段下降，约 -3.73 到 -3.83 percentage points。
- 这说明 EEG 的 M3 增强并非只来自单个 SEEG archetype，而更像是多个深部状态共同映射到同一个头皮 EEG 表达方向。

**Yeo17 解读边界**
- 当前 `Yeo17 delta heatmap` 更适合作为 archetype 空间指纹的解释图，而不是强 stage-dynamic 结果。
- 同一 archetype 的 Yeo17 loading delta 在 pre、LVFA、SZ/ictal、post 中高度一致，提示其主要反映 archetype 固有空间模式，而不是发作阶段导致的网络重构。
- 因此论文中应把 Yeo17 结果放在 secondary/explanatory 层级，用于解释 archetype 的网络偏好，而不要作为发作阶段演化的主要证据。

**总体解释**
- 当前结果更支持“同一 IDE-A-derived 状态空间中的发作阶段偏移”，而不是“发作阶段产生全新状态类别”。
- EEG 表层表现以 Microstate 3 增强为主；SEEG 深部表现以 dwell-time 动力学改变为主；二者在 SZ/ictal 与 post 阶段的关系共同向 M3 偏移。
- 该结果可被表述为 seizure evolution 中的 state redistribution / temporal reweighting，但不应单独解释为癫痫活动的因果传播链。

### 2.10 Statistical strategy
- 核心推断：patient-first
- 时间结构检验：circular-shift 或 block surrogate（2048 次置换）
- 组水平推断：sign-flip permutation（2048 次）
- 多重比较：Benjamini-Hochberg FDR
- 报告格式：mean effect、p_perm、q_fdr、n_subjects

### 3.7 IDE-A baseline reveals seizure-stage deviations in the same state space
**扩展分析 / 临床补充结果**

**主图建议：Figure 7**

- IDE-A、pre、LVFA、SZ、post 的分析流程图
- stage-wise 轨迹图
- delta heatmap
- paired effect size forest plot
- duration / confidence / denominator QC 作为 Figure 7 的前置支持面板

**结果展示原则：**
- 报告每个指标族的 matched patient 数
- 所有 stage 变化均相对于 IDE-A 计算 patient-level delta
- 优先展示 occupancy、dwell time 等一级核心指标
- 次级展示 Yeo17 loading 及 EEG-SEEG relationship 的 stage-wise 偏移
- EEG-SEEG relationship 明确作为三级探索指标，不与 primary delta 等权表述

**当前主要结果：**
- 主分析 denominator 支持充分：IDE-A、pre、LVFA、SZ/ictal、post 均有 20 名匹配患者完成 patient-first 配对比较。
- EEG microstate 的主要偏移集中在 Microstate 3：相对于 IDE-A，M3 occupancy 在 pre、LVFA、SZ/ictal、post 分别约增加 +3.79、+4.20、+6.18、+6.71 percentage points。
- post 阶段 EEG M3 dwell time 增加约 +50.0 ms，且 Microstate 2 occupancy 下降约 -3.15 percentage points，提示 postictal EEG 状态从 M2 向 M3 偏移。
- SEEG archetype occupancy 变化较小，多数 delta 在 +/-2 percentage points 内；更主要的变化发生在 dwell time。
- SEEG dwell time 在 pre/LVFA/SZ 阶段以缩短为主，尤其 pre A1 约 -20.1 ms、LVFA A1 约 -14.4 ms、SZ A1 约 -20.6 ms、SZ A3 约 -22.4 ms。
- post 阶段 SEEG dwell time 转为延长，A0 约 +26.8 ms、A2 约 +26.3 ms、A3 约 +19.2 ms。
- EEG-SEEG relationship 的主要方向是多个 SEEG archetype 到 EEG M3 的条件概率升高，最明显为 SZ A3 -> M3 约 +7.08 percentage points，post A3 -> M3 约 +7.34 percentage points。
- Yeo17 loading 当前更应解释为 archetype 的空间指纹，而不是强 stage-wise 动态证据，因为同一 archetype 的网络 loading delta 在不同 seizure stage 中高度一致。

**结果解读：**
发作阶段相对于 IDE-A 的改变主要表现为同一状态空间中的参数重加权，而非新状态类型的出现。EEG 表层状态以 M3 增强为主，SEEG 深部状态则以 dwell-time 动力学改变为主：发作前至发作期状态停留时间缩短，postictal 阶段停留时间延长。EEG-SEEG 条件概率结果进一步提示，SZ/ictal 与 post 阶段多个 SEEG archetype 更容易共同映射到 EEG M3，说明 M3 增强可能代表发作相关全局状态表达，而不是某一个深部 archetype 的一一对应投射。

**解释边界：**
该分析回答的是“同一状态空间在发作发展过程中如何偏移”，其价值在于连接静息态机制与癫痫阶段变化；但不应单独表述为发作因果机制模型。

## 4. Discussion

### 4.1 SEEG global-field states as an intracranial analog of EEG microstates
- 与 EEG microstate 的 dwell time、occupancy 对比[^1][^2]
- 病人内聚类与群体模板的差异
- 这是首次系统性在 SEEG 中提取全局场状态

### 4.2 Group archetypes: cross-patient reproducibility despite electrode heterogeneity
- Yeo17 投影如何解决电极异质性问题
- 中等 assignment similarity 的意义
- 网络偏好与功能解读[^9]

### 4.3 Near-zero-lag synchrony: shared drive over unidirectional propagation
- 41 ms 宽同步峰的神经生理解释
- 与 shared global drive 假说的关系[^6]
- 与 fMRI-EEG microstate 关联研究的对照[^7]

### 4.4 The similarity of archetype-conditioned EEG topographies
- 抹平效应、共享背景、方法学分辨度三种解释
- single-subject 结果的必要性
- 与同步结果的一致性

### 4.5 No transition-level driving: implications and interpretations
- switching 阴性的多重解释
- 样本量与功效限制
- 同步 vs 驱动的理论区分

### 4.6 Clinical implications and future directions
- SEEG field-state 分析框架在癫痫发作前状态检测中的潜在应用
- IDE-A vs SZ 阶段 delta 可作为后续预测模型特征，但需独立队列验证
- 向麻醉监测、认知状态监测等方向扩展

### 4.7 Limitations
- 样本量对 switching 功效的影响
- 癫痫患者样本的外推限制
- seizure-stage 分析的重复发作与阶段长度差异
- archetype assignment 的中等强度和不确定性
- K=4 选择仍需敏感性支持
- 宽带分析未区分频段贡献

## 5. Conclusions

SEEG 中存在离散的全局场状态，这些状态在共同功能网络空间中形成可重复的群体原型，并通过近零时滞的宽同步带与 EEG 微状态相联系。这种同步更符合共享全局驱动假说，而非单向状态传递机制。本研究建立了一套利用颅内记录研究大脑全局状态组织的跨模态分析框架，为理解 EEG 微状态的颅内神经基础提供了直接证据。
