# 科研路线与论文大纲_改进3

## 一、研究核心价值

**核心问题**：SEEG 中是否存在类似 EEG microstate 的离散全局场状态？这些状态能否在跨患者层面形成可重复原型？它们与 EEG microstate 的时间关系是什么？

**原创性**：
1. **填补空白**：EEG microstates 研究成熟[^1][^2]，但颅内对应物几乎空白
2. **方法学创新**：通过"患者内 field-state 提取 → 共同空间 archetype 投影"解决 SEEG 跨患者比较难题
3. **机制发现**：近零时滞宽同步峰（~41 ms）支持 shared global drive 假说，而非单向传递[^3]
4. **临床扩展**：发作阶段相对 IDE-A 基线的状态偏移揭示癫痫演化中的网络动力学重构

---

## 二、论文结构与图表配置

### 标题
**Discrete Global-Field States in Human SEEG: Identification, Group Archetypes, and Near-Zero-Lag Synchrony with EEG Microstates**

### Abstract（200-250词）
**Background**: EEG microstates 是反映大脑大尺度状态组织的离散头皮地形图。SEEG 中是否存在类似的全局场状态及其与 EEG microstate 的关系尚不清楚。

**Methods**: 分析 20 例患者的同步静息态 EEG-SEEG 记录。通过极性不变聚类从 SEEG 双极峰图提取患者内全局场状态（K=4），投影至 Yeo17 网络空间形成群体 archetype。使用 4 ms 分辨率的精细时滞耦合分析评估跨模态同步性。

**Results**: 所有患者均可靠识别出 SEEG 全局场状态（占比 24-27%；停留时间 96-104 ms）。形成 4 个具有不同 Yeo17 网络负载的可重复群体 archetype。EEG microstate 与 SEEG archetype 呈现稳健的近零时滞同步（峰值 0 ms；21/21 时滞点显著；峰宽 ~41 ms），符合共享全局驱动而非单向状态切换。

**Conclusions**: SEEG 包含离散全局场状态，形成可重复群体原型，并通过近零时滞耦合与 EEG microstate 同步。这些发现建立了利用颅内记录研究大脑大尺度状态组织的跨模态框架。

## 三、Methods

### 3.1 参与者与数据采集
- 20 例接受 SEEG 监测的癫痫外科评估患者
- 同步采集静息态 EEG（19 导联）和 SEEG（双极参考）
- 主分析：IDE-A 状态（发作间期静息态）
- 扩展分析：pre、LVFA、ictal、post 阶段

### 3.2 EEG 预处理与 microstate 标注
- 1-40 Hz 带通、250 Hz 重采样、平均参考
- 使用预训练 ModKMeans 模板（K=4）标注
- 输出：microstate 标签、GFP trace、GFP peaks

### 3.3 SEEG 预处理与双极道筛选
- 1-40 Hz 带通、250 Hz 重采样
- 仅保留两端接触点归属同一 AAL3 区域的双极道
- 患者内 z-score 标准化

### 3.4 患者内 SEEG global-field state 提取
- **全局峰值定义**：跨双极道 RMS 的局部最大值
- **峰图提取**：峰值时刻的瞬时双极空间构型
- **聚类**：极性不变模板聚类
- **K 值选择**：主分析保留 K=4，基于以下验证：
  - GEV 在 K=4 处达到 plateau
  - Incremental gain (K=4→5) 显著下降
  - Split-half stability 在 K=4 处最优
  - K=4 与经典 EEG microstate 文献一致
  - 完整模型阶数分析（K=2..7）见 Supplementary Figure S1
- **连续 backfitting**：最短状态持续时长 30 ms

### 3.5 共同空间群体 archetype 推导
- **共同空间**：Yeo17 网络分区
- **投影流程**：患者内模板 → Yeo17 网络均值 → 符号定向 → 跨患者 archetype matching
- **输出**：archetype 模板、support、assignment similarity
- **置换基准**：通过患者内模板标签置换建立 null 分布

### 3.6 精细时滞同步分析
- **分辨率**：250 Hz 原生时间轴
- **Lag 扫描**：[-40, +40] ms，步长 4 ms
- **效应量**：archetype 与 EEG microstate 的一致性指标
- **Null 分布**：circular-shift surrogate（2048 次）
- **组水平推断**：patient-first → sign-flip permutation（2048 次）

### 3.7 Archetype-conditioned EEG topography
- 将 archetype 标签对齐到 EEG 时间轴
- 计算每位患者每个 archetype 下的 EEG 平均 scalp map
- 汇总到 group-level 和 single-subject level
- 计算与 EEG microstate 模板的空间相似性
- 计算条件概率 P(EEG state | archetype)

### 3.8 IDE-A baseline vs seizure-stage 比较
- **参考空间**：所有发作阶段投影到 IDE-A-derived 状态空间
- **统计单位**：patient-first delta（患者内配对差值）
- **主效应量**：Δ(stage) = x̄_patient,stage - x̄_patient,IDE-A
- **指标优先级**：
  - 一级：occupancy、dwell time
  - 二级：Yeo17 loading 变化
  - 三级：EEG-SEEG 条件概率变化
- **敏感性分析**：CPS/GTCS 分层、重复发作控制

### 3.9 统计策略
- **核心推断**：patient-first
- **时间结构检验**：circular-shift surrogate
- **组水平推断**：sign-flip permutation
- **多重比较**：Benjamini-Hochberg FDR
- **报告格式**：mean effect、p_perm、q_fdr、n_subjects

## 五、Discussion

### 5.1 SEEG global-field states 作为 EEG microstate 的颅内类比
- 与 EEG microstate 的 dwell time、occupancy 对比[^1][^2]
- 首次系统性在 SEEG 中提取全局场状态

### 5.2 群体 archetype：跨患者可重复性
- Yeo17 投影解决电极异质性问题
- 置换基准证明 archetype 非随机聚类产物
- 中等 assignment similarity（0.348-0.536）的意义：反映真实个体差异

### 5.3 近零时滞同步：共享驱动而非单向传播
- 41 ms 宽同步峰的神经生理解释
- 与 shared global drive 假说的关系[^3]
- 与 fMRI-EEG microstate 关联研究的对照[^4]

### 5.4 Archetype-conditioned EEG topography 的高相似性
- 三种解释：群体平均抹平效应、共享 EEG 背景、方法学分辨度限制
- Single-subject 结果支持共享背景假说
- 与近零时滞同步结果的一致性

### 5.5 Switching 阴性结果的解释
- 样本量与功效限制
- 同步 vs 驱动的理论区分
- 支持 shared drive 而非 gating 模型

### 5.6 发作阶段的状态空间偏移
- 同一状态空间中的参数重加权，而非新状态类型
- EEG M3 增强 + SEEG dwell-time 动力学改变
- 临床意义：连接静息态机制与癫痫演化

### 5.7 局限性
- 样本量对 switching 功效的影响
- 癫痫患者样本的外推限制
- Seizure-stage 分析的重复发作与阶段长度差异
- K=4 选择的敏感性需进一步验证
- 宽带分析未区分频段贡献

### 5.8 未来方向
- 扩展至麻醉监测、认知状态监测
- 发作预测模型的特征工程
- 多频段分解分析

## 六、Conclusions

SEEG 中存在离散的全局场状态，这些状态在共同功能网络空间中形成可重复的群体原型，并通过近零时滞的宽同步带与 EEG 微状态相联系。这种同步更符合共享全局驱动假说，而非单向状态传递机制。发作阶段分析揭示了同一状态空间中的参数重加权模式。本研究建立了利用颅内记录研究大脑全局状态组织的跨模态分析框架，为理解 EEG 微状态的颅内神经基础提供了直接证据。
