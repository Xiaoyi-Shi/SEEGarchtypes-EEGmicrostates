# 一、研究价值评估

## 1.1 这个研究问题有多大价值？

**结论先行：这是一个高度原创、具有明确发表价值的研究问题，但需要谨慎控制叙事边界。**

### 支撑理由

**① EEG microstates是成熟领域，但颅内对应物几乎是空白**

EEG microstates自Lehmann等人1987年提出以来已有大量研究[^1]，在静息态、认知、疾病等方向均有发表[^2][^3]。然而，几乎所有microstate研究都局限在头皮EEG，原因很简单：颅内记录覆盖范围有限，且不同病人的电极位置各异，使得跨病人比较极具挑战性。你的研究通过"病人内field-state → 共同空间archetype投影"这一路线，正面解决了这个核心障碍。

**② SEEG的全局场状态研究是真正的空白点**

颅内脑电（iEEG/SEEG）的现有研究主要集中在：局部场电位（LFP）的频率特征、癫痫网络的传播分析、以及睡眠相关的慢波/spindle检测[^4][^5]。在静息态SEEG中定义类似microstate的"全局场状态"，目前文献中几乎没有系统性工作。这意味着你的Aim 1本身就是原创贡献。

**③ EEG-SEEG跨模态同步的近零时滞发现具有机制意义**

近零时滞的宽同步峰（peak width ~41 ms）这一发现，与当前关于大脑全局状态由共享驱动（shared global drive）维持的假说高度吻合[^6]。这不是一个平凡的阴性结果，而是一个机制性发现：EEG的全局场状态和SEEG的全局场状态之间，存在同步而非因果领先关系。这本身就是一个有理论价值的结论。

**④ Archetype方法学具有跨领域推广价值**

将病人内模板投影到共同空间（Yeo17）再做archetype matching，这一方法学本身对其他颅内记录研究（如DBS、皮层ECoG）都有参考价值。

# 二、当前路线的不足与改进建议

## 2.1 方法学层面的待强化点

### 问题一：为什么4个archetype的group-level EEG maps如此相似（r=0.982-0.999）？

这是当前最需要解释的结果，有三种可能解释，需要在论文中明确区分：

- **可能A（抹平效应）**：群体平均在不同archetype激活时的EEG样本，但由于每个archetype都发生在各种EEG背景下，平均后趋同。→ 需要做病人内single-subject archetype-conditioned maps，看是否也高度相似。
- **可能B（archetype本身对应相同EEG状态）**：SEEG的不同空间构型对应的EEG状态实际上相同，说明EEG微状态对SEEG内部差异不敏感。→ 这是一个有意思的阴性结果，需要正面论述。
- **可能C（方法学confound）**：archetype assignment similarity中等（0.348-0.536），说明archetype本身的跨病人一致性不高，投影后的group map可能都在向"平均模板"方向退化。

**建议**：
- 增加single-subject层面的archetype-conditioned EEG maps可视化
- 在论文Discussion中明确讨论这三种解释，不要回避

### 问题二：近零时滞宽同步峰（41 ms）的解释需要更严格的对照

宽同步峰可能的来源：
- 真实的生理同步（最期望的解释）
- EEG和SEEG都受共同上游驱动（GFP控制后已经部分排除）
- 方法学artifact（backfitting平滑导致的状态标签时间相关）

**建议**：
- 增加一组关键对照：用不同带宽（如1-20 Hz vs 1-40 Hz）重跑，看峰宽是否改变
- 明确报告backfitting的最小时长（30 ms）对峰宽估计的影响
- 考虑用独立的microstate分割方法（如AAHC而非ModKMeans）做鲁棒性检验

### 问题三：Archetype matching强度中等（0.348-0.536）需要统计基准

这些相似度数值本身没有自然的"好坏"参照系。

**建议**：
- 增加随机基准：如果完全随机分配病人内模板，archetype similarity期望值是多少？
- 可以通过置换病人内模板标签（不是时间轴置换，而是病人间随机混合）生成null分布，报告观测值的百分位
- 这一分析会大幅强化Aim 2的说服力

### 问题四：SEEG field-state的K=4选择需要更多验证

目前固定K=4，与EEG microstate的经典4状态对应，这个选择需要被论证，而不是默认。

**建议**：
- 增加K选择的敏感性分析（当前工程已支持 supplementary `K=2..7`）
- 不要只给单一肘图，至少同时报告 template fit、incremental gain、split-half stability
- 说明K=4既有数据驱动支持，也与EEG microstate文献对齐

### 问题五：受试者样本量（N=20）与统计功效

20例受试者对于病人内分析是合理的，但对于某些group-level效应检测可能功效不足，这可能部分解释了switching分析的阴性结果。

**建议**：
- 在局限性部分明确讨论样本量对switching检测功效的影响
- 可以做一个简单的功效分析：在当前effect size下，需要多少受试者才能稳定检测到switching效应？

## 2.2 叙事层面的改进建议

### 改进一：阴性结果需要被积极论述，而不是被动承认

当前路线把switching阴性结果定位为"机制边界"，这是正确的。但论文写作中还需要进一步强调：**近零时滞同步+无单向驱动**这个组合，本身就是一个有理论意义的发现。

这与以下领域的核心争论直接相关：
- Brain states是由局部→全局传播驱动，还是由全局共同驱动维持？
- EEG microstates究竟代表什么层次的神经活动？

你的数据支持"shared global drive"而非"单向传播"，这个立场在论文Discussion中应该得到充分展开。

### 改进二：强化与fMRI-EEG microstate关联研究的对比

fMRI与EEG microstate的关联研究是目前microstate跨模态比较的主流[^7]。你的SEEG研究提供了比fMRI更高时间分辨率的颅内记录，这一优势需要在Introduction中明确定位。

### 改进三：临床背景的定位

受试者来自癫痫外科评估队列（这是SEEG的典型来源），需要在方法中明确这一点，并在局限性中讨论：静息态数据是否在发作间期采集、是否排除了受监测区域的异常放电干扰。

---

## 2.3 值得补充的探索方向

### 补充方向一：SEEG field-state的频率特异性

当前使用1-40 Hz宽带信号。是否可以在特定频段（如alpha 8-13 Hz，theta 4-8 Hz）重复field-state提取？这会回答"SEEG field-state是否由特定节律主导"这一问题，也与EEG microstate研究中已知的alpha节律关联形成对话[^8]。

### 补充方向二：SEEG archetype的功能网络解读

当前archetype的Yeo17 loading pattern（如A2 LB-DA主要偏向LimbicA）有初步描述，但这些网络偏好是否具有生理意义，需要与已知的功能网络角色相联系。建议在Discussion中专门讨论这些loading pattern的可能神经生理含义。

### 补充方向三：与脑电微状态文献中的认知/临床关联对接

EEG microstate文献已有大量认知功能和临床意义的研究。你的研究目前是方法学/基础神经科学导向，可以在Discussion中指出SEEG field-state分析框架未来在癫痫网络研究、麻醉状态监测等方向的应用潜力。

# 三、完善后的科研路线与论文大纲

## 论文标题（建议）

**英文**：*Discrete Global-Field States in Human SEEG: Identification, Group Archetypes, and Near-Zero-Lag Synchrony with EEG Microstates*

**中文**：*人类SEEG中的离散全局场状态：识别、群体原型及其与EEG微状态的近零时滞同步*

---

## 完整论文大纲

---

### Abstract

**Background**：EEG microstates are discrete, quasi-stable global scalp topographies reflecting large-scale brain state organization. Whether analogous global-field states exist in stereoelectroencephalography (SEEG) and how they relate to EEG microstates remains unknown.

**Methods**：We analyzed simultaneous resting-state EEG and SEEG recordings from 20 patients. Subject-level SEEG global-field states were extracted from bipolar peak maps using polarity-invariant clustering, with `K=4` retained for the main text and a supplementary `K=2..7` model-order evaluation used to quantify fit plateau, incremental gain, and stability. States were projected to a common Yeo17 network space to derive group archetypes. Synchrony between SEEG archetypes and EEG microstates was evaluated using fine-lag coupling at native 4-ms resolution.

**Results**：SEEG global-field states were reliably identified in all 20 patients (occupancy: 24–27%; dwell time: 96–104 ms). Four reproducible group archetypes emerged with distinct Yeo17 network loadings. EEG microstates and SEEG archetypes exhibited robust near-zero-lag synchrony (peak at 0 ms; 21/21 lag points significant; peak width ~41 ms), consistent with shared global drive rather than unidirectional state switching.

**Conclusions**：SEEG contains discrete global-field states that form reproducible group archetypes and synchronize with EEG microstates through near-zero-lag coupling. These findings establish a cross-modal framework for studying large-scale brain state organization using intracranial recordings.

### 1. Introduction

#### 1.1 EEG microstates and global brain state organization

- EEG microstates的发现历史与核心特征（Lehmann 1987以来的发展）[^1][^2]
- 四类经典microstate的功能关联与网络对应[^3]
- Microstate研究的时间分辨率优势与头皮空间分辨率局限
- 跨模态比较（fMRI-EEG）的现有成果与争议[^7]

#### 1.2 SEEG的特点与现有全局场分析

- SEEG作为癫痫外科评估工具的独特优势：真实神经元活动、高时空分辨率
- 现有SEEG分析的局限：多聚焦于局部、癫痫事件、频率特征
- 颅内EEG的global field power概念的引入与有限应用[^4][^5]
- SEEG全局场状态研究的空白

#### 1.3 EEG-SEEG跨模态状态对应：尚未回答的问题

- 核心问题提出：SEEG中是否存在类似EEG microstate的离散全局场状态？
- 若存在，这些状态是否能在跨病人层面形成可重复的群体archetype？
- EEG microstate与SEEG field-state的时间关系：同步还是单向驱动？

#### 1.4 本研究的策略与贡献

- 病人内field-state提取 → 共同空间archetype投影 → 跨模态精细时滞同步分析
- 方法学创新：解决SEEG跨病人比较难题的archetype框架
- 填补EEG-SEEG状态对应关系研究的空白

### 2. Methods

#### 2.1 参与者与数据采集

- 受试者：20例接受SEEG监测的癫痫外科评估患者
- 数据：同步采集的resting-state EEG（11导联）和SEEG（双极参考）
- 分析时间段：IDE_A状态（来自workbook标注）
- 排除标准：数据质量、有效双极道数量、EEG覆盖完整性

#### 2.2 EEG预处理与microstate标注

- 信号预处理：1-40 Hz带通、250 Hz重采样、标准19导布局插值、平均参考
- Microstate模板：使用预训练ModKMeans模板（K=4），直接标注而非重新拟合
- 输出：每时间点的microstate标签、GFP trace、GFP peaks
- 方法参考[^2][^3]

#### 2.3 SEEG预处理与双极道筛选

- 双极信号：1-40 Hz带通、250 Hz重采样
- 严格same-region筛选：仅保留两端接触点均归属同一AAL3区域的双极道
- 病人内z-score标准化（通道级）

#### 2.4 Subject-level SEEG global-field state提取

- 全局峰值定义：跨双极道RMS的局部最大值
- 峰图提取：在峰值时刻提取瞬时双极空间构型
- 聚类：极性不变模板聚类，主分析 retained `K=4`
- 连续backfitting：最小状态持续时长30 ms
- K值支持依据：supplementary `K=2..7` model-order evaluation（fit plateau、incremental gain、split-half stability、interpretability）+ 与EEG microstate文献对齐

#### 2.5 Group archetype derivation in common space

- 共同空间：Yeo17网络分区
- 投影步骤：病人内模板→Yeo17网络均值→符号定向→跨病人archetype matching
- 输出：archetype模板、support、assignment similarity
- 重要解释边界：Yeo17负载值为相对空间贡献，非绝对生理幅度

#### 2.6 Fine-lag synchrony analysis

- 分析网格：共享250 Hz
- Lag扫描窗口：[-40, +40] ms，步长4 ms（原生分辨率）
- 效应量：archetype-EEG microstate之间的一致性指标
- Null分布：circular-shift surrogate（保持时间结构）
- Patient-first → group-level sign-flip permutation

#### 2.7 Archetype-conditioned EEG topography

- Archetype标签对齐到EEG传感器时间轴
- 每病人每archetype的平均scalp map → group-level汇总
- 与EEG microstate模板的空间相似性比较
- 条件概率P(EEG state=i | archetype=j)

#### 2.8 Supplementary analyses

- GFP-global coupling（主要supplementary结果）
- GFP控制后的microstate/transition效应（验证GFP混淆）
- SEEG→EEG switching分析（机制边界检验）

#### 2.9 IDE-A baseline vs seizure-stage comparison

- 比较框架：`IDE_A`作为患者内基线，发作阶段按`pre → LVFA → SZ → post`排序。
- 状态空间：不在SZ阶段重新聚类；EEG microstate与SEEG archetype均投影/标注到IDE-A-derived reference space，保证状态编号可比。
- 统计单位：主分析使用patient-first summaries；同一患者的多次发作先在患者内按stage求均值，seizure-level行仅用于追踪与后续混合模型。
- 核心效应量：`delta(stage) = patient_mean(stage) - patient_mean(IDE_A)`。
- 指标族：EEG microstate参数、SEEG archetype参数、Yeo17网络loading、EEG-SEEG relationship条件概率。
- 分层变量：保留`seizure_type`（如CPS/GTCS）用于补充分析，但主图先展示全体患者的配对delta。
- 解释边界：该分析描述发作阶段相对IDE-A的状态偏移，不能单独证明癫痫发展或发作演化的因果机制。

#### 2.10 统计策略

- 所有核心推断：patient-first
- 时间结构检验：circular-shift或block surrogate（2048次置换）
- 组水平：sign-flip permutation（2048次）
- 多重比较：Benjamini-Hochberg FDR
- 报告格式：mean effect、p_perm、q_fdr、n_subjects

**(Aim 3的主要结果——论文核心图)**

**主图（Figure 4）**：

#### 3.4 Archetype-conditioned EEG topographies reflect a shared EEG background pattern

**(Aim 4的主要结果)**

**主图（Figure 5）**：

- GFP控制后的microstate效应（阴性）

**主要结果陈述**：
- EEG GFP与SEEG global magnitude在0 ms处显著耦合（mean effect = 0.124，q_fdr = 0.000488）
- GFP peak周围的SEEG轨迹显示宽正偏，约+36 ms处有轻微负偏
- 控制GFP后，microstate omnibus不显著（q_fdr = 0.1196）

**解释**：两种模态的全局振幅存在强共同驱动。这种共享global drive是EEG-SEEG同步的重要基础机制，但microstate标识本身（超越GFP振幅的状态身份信息）对SEEG全局振幅的额外解释力有限。

#### 3.7 Exploratory extension: seizure stages deviate from the IDE-A baseline

**(临床/发作阶段扩展结果，不替代主论文的静息态机制结果)**

**主图/扩展图（Figure 7）**：
- IDE-A、pre、LVFA、SZ、post的metric-family denominator/QC图
- EEG microstate occupancy/dwell/occurrence相对IDE-A的stage-wise delta
- SEEG archetype occupancy/dwell/assignment similarity相对IDE-A的stage-wise delta
- `archetype × Yeo17 network × stage` signed loading-delta heat图
- `SEEG archetype × EEG microstate × stage` relationship signed-delta heat图

**主要结果陈述方向**：
- 报告每个metric family的matched patient数，避免把99次发作当作99个独立受试者。
- EEG microstate结果重点描述哪些microstate在LVFA/SZ/post阶段相对IDE-A占比升高或降低。
- SEEG archetype结果重点描述哪些archetype在发作阶段出现occupancy、dwell time或assignment similarity偏移。
- Yeo17结果重点描述哪些功能网络在特定archetype中相对IDE-A呈持续升高或降低。
- EEG-SEEG relationship结果重点描述发作阶段是否强化或削弱特定`archetype → microstate`条件概率。

**解释**：这一扩展把原本的IDE-A参考状态空间用于发作阶段比较，回答“同一套EEG microstate/SEEG archetype框架在发作发展过程中如何偏移”。它的价值在于连接基础机制结果与癫痫发作阶段，但写作中必须强调这是观察性、患者内配对的stage deviation分析，不应表述为单向因果或疾病进展模型。

### 4. Discussion

#### 4.1 SEEG global-field states as an intracranial analog of EEG microstates

- 与EEG microstate的dwell time（典型60-120 ms）和occupancy（约25%/状态）对比[^1][^2]
- 方法学差异的讨论：双极参考 vs 头皮参考，病人内聚类 vs 群体模板
- 这是文献中首次系统性地在SEEG中提取和描述全局场状态

#### 4.2 Group archetypes: cross-patient reproducibility despite electrode heterogeneity

- Yeo17投影策略如何解决颅内记录的空间异质性问题
- 中等assignment similarity的意义：真实神经异质性 vs 方法学局限
- Archetype的网络解读：Limbic/SalVentAttn/Default网络的功能角色[^9]

#### 4.3 Near-zero-lag synchrony: shared drive over unidirectional propagation

- 41 ms宽同步峰的神经生理解释
- 与"Global neuronal workspace"理论的关系[^6]
- 与fMRI-EEG microstate关联研究的时间分辨率优势对比[^7]
- 为什么没有单向传播：从共享皮层-皮层下驱动的角度解释

#### 4.4 The similarity of archetype-conditioned EEG topographies

- 三种可能解释的详细讨论（抹平效应/共享背景/archetype本身分辨度）
- 单受试者层面的topography分析作为后续研究方向
- 这一发现与同步结果的一致性

#### 4.5 No transition-level driving: implications and interpretations

- Switching阴性的多种解释（效应量小、样本量功效不足、真实无效）
- 与fMRI研究中microstate transition关联的对比
- 同步 vs 驱动：这一区分对脑状态理论的意义

#### 4.6 Clinical implications and future directions

- SEEG field-state分析框架在癫痫发作前状态检测中的潜在应用
- IDE-A vs SZ阶段delta可作为后续发作预测/分期模型的候选特征，但需要独立队列验证
- 麻醉深度监测、认知状态监测的扩展可能性
- 未来：更大样本量、频段特异性分析、与癫痫网络的关系

#### 4.7 Limitations

- 样本量（N=20）对switching检测功效的影响（附功效分析结果）
- 受试者为癫痫患者，结果向健康人群的推广需谨慎
- SZ阶段比较存在重复发作、阶段长度和CPS/GTCS组成差异，主分析必须使用patient-first delta并报告denominator
- Archetype assignment similarity中等，部分个体的archetype归属存在不确定性
- K=4的选择（虽有GEV支持，但不同K值下结果的敏感性需报告）
- 1-40 Hz宽带分析未区分频段贡献

---

### 5. Conclusions

SEEG中存在离散的全局场状态，这些状态在共同功能网络空间中形成可重复的群体原型，并通过近零时滞的宽同步带与EEG微状态相联系。这种同步关系与共享全局驱动假说一致，而非单向状态传递机制。本研究建立了一套利用颅内记录研究大脑全局状态组织的跨模态分析框架，为理解EEG微状态的颅内神经基础提供了直接证据。
