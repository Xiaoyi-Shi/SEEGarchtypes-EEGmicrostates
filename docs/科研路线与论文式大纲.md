# SEEG Global-Field States 与 EEG Microstates 研究路线及论文式大纲

## 1. 文档目的

这份文档用于统一后续三类工作：

- 明确整个项目最有科学价值的主研究问题
- 为工程精简和流程重构提供取舍依据
- 为论文方法、结果和图表写作提供可直接复用的结构

本文档以当前仓库截至 `2026-04-09` 的实现和结果为基础，目标不是罗列所有 exploratory 分支，而是收敛出一条最清晰、最稳、最可发表的主线。

## 2. 当前结果给出的最稳定信号

截至目前，最稳的结果不是区域均值差异，也不是区域连接差异，而是跨模态状态组织和全局背景同步。

### 2.1 主流程结论

- `AAL3` 和 `Yeo17` 主流程中的 activity / connectivity omnibus 大多为阴性
- `Yeo17` 下网络均值和网络对连接没有稳定的组水平 FDR 阳性结果
- 这说明“EEG state 条件下的静态区域均值差异”不是当前数据最敏感的层次

当前已验证的主流程结果：

- `AAL3` 主流程：`run-id = 20260407_083346`
  - `20/32` 名受试者进入主 cohort
  - activity omnibus 与 post-hoc 都没有 `q_fdr < 0.05`
  - connectivity omnibus 只有 `corr` 下 `1` 个区域对通过 FDR：
    `Right Hippocampus - Right Superior temporal gyrus`
    `n_subjects = 9`
    `statistic = 7.49`
    `p_perm = 0.000488`
    `q_fdr = 0.0400`
- `Yeo17` 主流程：`run-id = 20260407_092306`
  - activity omnibus：`13` 个网络进入组统计，显著数 `0`
  - connectivity omnibus：`corr / plv / wpli` 显著数都为 `0`
  - 最接近的是 `wpli` 下 `DefaultB - DefaultA`
    `p_perm = 0.00390`
    `q_fdr = 0.215`

### 2.2 GFP-informed global 分支

- `EEG GFP` 与 `SEEG global magnitude` 存在稳定共享耦合
- 这种耦合在 `0 ms` 附近最强，但峰形较宽，更像 shared global drive，而不是明确的单向传播
- 控制 `GFP` 后，whole-brain global 层面的 microstate / transition 效应明显减弱

当前已验证的 GFP/global 结果：

- `run-id = 20260407_155202`
- `Yeo17 core 11` 网络的主指标 `rms__equal`
- 连续 GFP-global coupling：
  `lag = 0 ms`
  `mean_effect = 0.124468`
  `p_perm = q_fdr = 0.000488`
- lagged GFP/global：
  峰值在 `0 ms`
  `mean_effect = 0.124158`
  `p_perm = 0.000488`
  `q_fdr = 0.002684`
  显著 lag 点数 `11`
- peak-centered trajectory：
  整个窗口以浅负偏为主
  最深约在 `+36 ms`
  `mean_effect = -0.012481`
  `p_perm = 0.000488`
  `q_fdr = 0.000684`
- 控制 GFP 后：
  - `gfp-controlled-microstate` omnibus 不显著
    `statistic = 1.916`
    `q_fdr = 0.1196`
  - `gfp-controlled-transition` 全部不显著

### 2.3 SEEG field-state 分支

- 病人内 `SEEG global-field states` 可以稳定提取
- occupancy 和 dwell time 没有塌缩，说明这些状态不是伪模板
- `SEEG field-state` 与 `EEG microstate` 在 `0 ms` 存在稳定同步耦合

当前已验证的 field-state 结果：

- `run-id = 20260407_202010`
- `20/20` 名受试者都成功生成病人内 `SEEG global-field states`
- 每人固定 `4` 个 states；每人有效双极道 `115-197` 条；峰图数 `3244-5000`
- 四个 states 平均 occupancy 约 `24%-27%`
- 平均 dwell time 约 `96-104 ms`
- `EEG microstate ↔ SEEG field-state` 同步耦合：
  `lag = 0 ms`
  `mean_effect = 0.00130`
  `p_perm = q_fdr = 0.000976`
- field-state transition coupling 没有过 FDR
- 控制 GFP 后的 field-state switching 也不显著

### 2.4 archetype 与精细时序分支

- 病人内 `SEEG field states` 投影到共同空间后，可以得到群体 `archetypes`
- `SEEG archetype` 与 EEG 的关系主要表现为近零时滞的宽同步峰
- `SEEG -> EEG switching` 目前没有稳定 FDR 阳性
- `archetype-conditioned EEG maps` 存在，但四个 archetype 对应的组水平 scalp maps 目前高度相似

当前已验证的 archetype / fine-lag / conditioned-topography 结果：

- archetype 与 fine-lag：`run-id = 20260408_083720`
  - `20/20` 名受试者支持 `4` 个 archetypes
  - archetype assignment similarity 为中等：
    `mean_similarity = 0.348-0.536`
  - `4 ms` fine-lag：
    `-40` 到 `+40 ms` 的 `21/21` 个 lag 点都过 FDR
    峰值在 `0 ms`
    `mean_effect = 0.001314`
    `p_perm = 0.00244`
    `q_fdr = 0.003843`
  - peak-lag 本身不显著偏离零：
    `mean = 5 ms`
    `median = 0 ms`
    `q_fdr = 0.434`
  - peak-width 显著：
    `mean = 41 ms`
    `q_fdr = 0.000732`
- archetype-conditioned EEG topography：
  首轮 exploratory `run-id = 20260408_102110`
  最新 paper-focused 重跑 `run-id = 20260408_131613`
  - 4 张组水平 `archetype-conditioned EEG maps` 两两相关为 `0.982-0.999`
  - 最佳 archetype-to-template 相似性仅 `0.170-0.192`
  - 偏好大致为：
    archetype `0/1/2 -> EEG state 0`
    archetype `3 -> EEG state 2`
  - 零时滞条件概率偏好全部不显著
    最低 `q_fdr` 仍大于 `0.39`
  - `SEEG -> EEG switching` 与 `archetype -> EEG switching` 都没有稳定 FDR 阳性

### 2.5 model-order supplementary 分支

- 主文当前继续保留病人内 `K=4` field-state 作为默认设置
- `K=4` 的保留不是因为单一经验偏好，而是因为论文叙事需要同时满足：
  - 模板拟合在较小 `K` 后继续提升但逐步进入平台
  - split-half 稳定性保持可接受
  - state occupancy 和病人支持不发生明显塌缩
  - 下游 archetype / synchrony 主线维持可解释性
- 当前工程已经加入 `K=2..7` 的 supplementary model-order 评估分支，并通过分类表格与 supplementary R figure family 输出
- 正文可以明确写：`K=4` is retained as the main-text default and is supported by fit plateau, stability, and interpretability, with multi-`K` evaluation reported in the Supplementary Materials

### 2.6 截至目前最稳的经验结论

- 最强信号是 `EEG GFP <-> SEEG global magnitude` 的共享全局驱动
- 第二稳的信号是 `EEG microstate <-> SEEG field-state / archetype` 的近零时滞宽同步峰
- 当前不支持把 `SEEG -> EEG switching` 或 `EEG -> SEEG switching` 写成主结论
- 当前也不支持把某个 `SEEG archetype` 强行解释成某个单一 `EEG microstate`

## 3. 建议的主研究问题

最适合当前项目的核心问题是：

**SEEG 是否也存在离散的 global-field states，这些 states 能否形成群体 archetypes，并与 EEG microstates 在近零时滞上稳定对齐。**

这是当前最稳、最可解释、也最能形成跨模态叙事的问题定义。

不建议将下面这些问题作为主论文 headline：

- `AAL3` / `Yeo17` 区域均值差异是否显著
- 某个 SEEG archetype 是否一一对应某个 EEG microstate
- `SEEG transition -> EEG transition` 是否稳定成立
- `EEG transition -> SEEG transition` 是否稳定成立

这些问题更适合作为补充分析或机制边界讨论。

## 4. 论文核心叙事

### 4.1 一句话版本

**Subject-level SEEG global-field states exist, form group archetypes in a common comparison space, and align with EEG microstates through near-zero-lag synchrony rather than strong transition driving.**

### 4.2 中文叙事版本

我们提出：SEEG 不仅包含连续的全局振幅变化，还可以在病人内被组织为若干离散的 global-field states；这些 states 在共同脑空间中呈现可重复的群体 archetypes，并与 EEG microstates 在近零时滞上呈稳定同步，而不是主要通过明确的状态切换驱动相互影响。

## 5. 核心假设

- `H1`：病人内可以从双极道 SEEG 峰图中提取稳定的 `global-field states`
- `H2`：这些病人内 states 在共同空间中存在可重复的群体 `archetypes`
- `H3`：`EEG microstates` 与 `SEEG field-state / archetype` 的耦合峰位于 `0 ms` 附近，并在原生 `4 ms` 分辨率下呈现宽同步峰
- `H4`：`SEEG -> EEG` 的切换关系若存在，其效应弱于同步本身，且不应预设为主结论

## 6. 论文 Aims

## Aim 1. 证明病人内存在 SEEG global-field states

### 科学问题

SEEG 是否不仅仅表现为连续振幅变化，而是具有离散、可重复的瞬时空间构型。

### 当前证据

- 病人内稳定得到 `4` 个 field states
- 各 state 的 occupancy 和 dwell 较均衡
- 没有塌缩到单一模板
- 最新 paper-focused 重跑 `run-id = 20260408_131613`
  中 4 个 states 的平均 occupancy 为：
  `0.246 / 0.241 / 0.239 / 0.273`
  平均 dwell time 为：
  `96.5 / 103.4 / 104.2 / 102.2 ms`

### 方法学定义

- 输入：病人内裁剪后的双极道 SEEG
- 频带：`1-40 Hz`
- 采样率：`250 Hz`
- 标准化：每个双极道病人内 `z-score`
- 峰值定义：跨双极道 `RMS`
- 峰图：在 `SEEG global metric` 峰值时刻提取瞬时 bipolar map
- 聚类相似度：极性不变 `|corr|`
- 状态数：主分析 retained `K=4`
- supplementary model-order supporting analysis：`K=2..7`
- 回填：continuous backfitting
- 最小时长：`30 ms`

### 主结果指标

- state occupancy
- dwell time
- state transition matrix
- 模板拟合稳定性
- 病人内重复性
- supplementary model-order support：fit plateau、incremental gain、split-half stability、interpretability-collapse diagnostics

### 论文中的作用

这是整篇文章的基础层，回答“SEEG 中有没有类似 EEG microstate 的离散场状态”。

## Aim 2. 证明这些病人内 states 在共同空间中具有群体 archetypes

### 科学问题

病人内定义的 `SEEG field states` 能否在跨病人的共同空间中表现为可重复的 archetypes，而不只是完全个体化的模板。

### 当前证据

- 所有 `20/20` 受试者都能支持 4 个 archetypes
- archetype assignment similarity 为中等，不是很高，但稳定存在
- 4 个 archetype 的 `mean_similarity` 为：
  `0.348 / 0.536 / 0.491 / 0.470`
- 当前群体空间模式大致表现为：
  - archetype 0：`SalVentAttnB + TempPar + DefaultA`
  - archetype 1：`LimbicA + SalVentAttnB + SomMotB`
  - archetype 2：`LimbicA` 最突出
  - archetype 3：`ContA + SalVentAttnA + LimbicB`

### 方法学定义

- 比较空间：主分析使用 `Yeo17`
- 病人内模板先投影到共同空间
- 做 sign orientation
- 做 subject-to-archetype matching
- 输出 archetype template、support、assignment similarity

### 主结果指标

- archetype support
- assignment similarity
- archetype reproducibility
- archetype spatial loading pattern

### 论文中的作用

这是从“病人内存在”升级到“群体共性存在”的关键桥梁，也是后续跨模态比较的前提。

### 写作边界

- 不要直接把 archetype 写成“某个 AAL3 区域高、某个区域低”的主结论
- 主结果应该先用 `Yeo17` 讲群体共性
- `AAL3` 更适合补充材料做细空间解释

## Aim 3. 检验 SEEG archetype 与 EEG microstate 的近零时滞同步

### 科学问题

`SEEG field-state archetype` 与 `EEG microstate` 的关系是否主要表现为同步状态组织，而不是明确的单向领先或滞后。

### 当前证据

- `0 ms` 附近存在稳定正耦合
- 原生 `4 ms` fine-lag 扫描显示整段近零窗口为阳性
- 但峰较宽，不是尖锐单点锁定
- 最新 paper-focused 重跑中：
  `21/21` 个 lag 点都显著
  峰值在 `0 ms`
  `mean_effect = 0.001314`
  `q_fdr = 0.003843`
  peak-width 约 `41 ms`

### 方法学定义

- EEG 时间轴与 SEEG 时间轴都使用共享 `250 Hz` 分析网格
- lag 扫描窗口：主分析建议 `[-40, +40] ms`
- lag 步长：原生 `4 ms`
- patient-first 计算 archetype 与 EEG microstate 的耦合
- 结构化 null：circular shift 或 block-preserving surrogate

### 主结果指标

- `lag_ms`
- mean effect
- group-level peak lag
- peak width
- subject-level peak-lag distribution

### 结果解释原则

- 如果峰位于 `0 ms` 附近且宽度较大，解释为 near-zero-lag synchrony
- 不要过度写成精确锁定或源-传递顺序
- 如果个体 peak lag 分布分散，而组水平宽峰稳定，说明是群体共同同步带，而不是严格固定时延

### 论文中的作用

这是整篇文章最可能成为主结果图的部分，因为它最直接体现 EEG-SEEG 跨模态状态关系。

## Aim 4. 描述 archetype-conditioned EEG topography，并评估是否存在 archetype-to-microstate preference

### 科学问题

当某个 `SEEG archetype` 活跃时，EEG 头皮图是否呈现稳定的 scalp topography；这些 topographies 是否偏向某个 EEG microstate 模板。

### 当前证据

- archetype-conditioned EEG maps 已经可以导出
- 目前 4 个组平均 scalp maps 高度相似
- 与 EEG microstate 模板的空间相似性和零时滞偏好都存在，但较弱且分布式
- 4 张 group scalp maps 两两相关达到 `0.982-0.999`
- archetype-to-template 最佳相似性只有 `0.170-0.192`
- 条件概率偏好未过 FDR

### 方法学定义

- 先用 archetype assignment 将病人内 `field_state` 映射到群体 archetype
- 在共享时间轴上对齐到 staged EEG 19 导传感器样本
- 对每个病人、每个 archetype 求平均 EEG scalp map
- 再做组水平平均
- 与 staged EEG microstate template 做空间相似性比较
- 输出 `P(EEG state=i | archetype=j)` 一类的偏好摘要

### 主结果指标

- archetype-conditioned group scalp maps
- archetype x EEG template similarity matrix
- archetype x EEG state conditional probability matrix
- distributed preference diagnostics

### 结果解释原则

- 如果多个 EEG templates 同时具有相近相似性，不应强行做一一映射
- 如果 group scalp maps 彼此高度相似，要优先解释为共同 EEG 背景模式，而不是 4 个截然不同的头皮模板

### 论文中的作用

这一部分最适合作为“状态同步具体表现为何种 EEG scalp pattern”的桥接层。

## Aim 5. 将 switching 分析作为机制边界，而不是主 headline

### 科学问题

`SEEG field-state transition` 后，EEG 是否更容易进入某个 microstate 或发生 switching。

### 当前证据

- `SEEG -> EEG switching` 没有稳定 FDR 阳性
- 说明当前数据不支持把切换门控作为主结论
- 最新 paper-focused 重跑中：
  - `field-state -> EEG switching` 最低 `q_fdr = 0.293`
  - `archetype -> EEG switching` 最低 `q_fdr = 0.609`
  - `GFP-controlled field-state -> EEG switching` 最低 `q_fdr = 0.452`

### 方法学定义

- 定义 `SEEG field-state transition` 事件
- 在后续窗口内统计 EEG state entry 或 any-switch
- patient-first
- temporal-structure-preserving null

### 写作策略

- 放在次级结果或补充材料
- 用来限定主结论边界
- 结论应写成“未见稳定组水平 transition-driving evidence”

## 7. 论文方法学草案

### 7.1 受试者与数据

- 受试者来自 IDE resting-state 数据
- 主分析状态可固定为 `IDE_A`
- EEG 与 SEEG 都裁剪到 workbook 标注的同一分析时间窗
- 群体纳入标准沿用当前 cohort 过滤逻辑

### 7.2 EEG 预处理

- 共享 11 导联输入：`F3/Fz/F4/C3/Cz/C4/P3/Pz/P4/O1/O2`
- 带通：`1-40 Hz`
- 重采样：`250 Hz`
- 扩展到标准 `19` 导布局并做插值恢复
- 平均参考
- 使用预置 `ModKMeans` 模板标注 `EEG microstates`
- 输出 `microstate labels`、`corr`、`GFP trace`、`GFP peaks`

### 7.3 SEEG field-state derivation

- 直接使用裁剪后的双极道 SEEG，不先做 AAL3/Yeo17 均值
- 病人内每个双极道 `z-score`
- 使用 `RMS` 定义全局峰值
- 在峰值时刻提取瞬时 bipolar map
- 采用极性不变的模板相似度
- 聚类得到病人内 `K=4` field states
- 全时序 backfitting

### 7.4 群体 archetype 投影

- 将病人内模板投影到 `Yeo17`
- 进行 sign orientation
- 进行 subject-to-archetype matching
- 计算 support、similarity 和 archetype loadings

重要解释边界：

- `Yeo17` 列值不是原始振幅或功率
- 这些值来自病人内双极道标准化后模板在共同空间中的相对空间负载
- 具体做法是：
  病人内通道先做 `z-score`
  再构造零均值、单位范数的空间模式
  投影到共同 `Yeo17` 空间后再做模式标准化和符号定向
- 因此：
  正值表示该网络在该 archetype 中相对更高
  负值表示该网络在该 archetype 中相对更低
  绝对值大小表示该网络对该 archetype 区分的贡献强弱
- 不应把这些值解释成绝对生理功率，也不应把正负号直接解释为兴奋/抑制

### 7.5 archetype-conditioned EEG topography

- 将 archetype label 对齐到 EEG 传感器时间轴
- 对每个病人、每个 archetype 计算 EEG 平均 scalp map
- 组水平聚合为 group archetype-conditioned maps
- 与 EEG microstate templates 比较空间相似性

### 7.6 fine-lag synchrony

- 在共享 `250 Hz` 网格上进行近零时滞扫描
- 窗口建议 `[-40, +40] ms`
- 步长固定 `4 ms`
- 输出 group lag curve 与 subject peak summary

### 7.7 统计策略

- 所有核心推断使用 patient-first
- 不做 pooled sample inference
- 对于 lag / transition / preference 等时间结构问题，使用 circular-shift 或 block surrogate
- 组水平使用 sign-flip permutation
- 多重比较使用 FDR
